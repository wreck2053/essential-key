On-device setup

Remap the Essential Key without a PC.

No desktop ADB installation or terminal commands are required. Android still requires you to explicitly enable and approve Wireless Debugging.

What “without a PC” means

The app contains a local ADB client. It discovers Android's Wireless Debugging service on the phone, pairs using the six-digit code shown by Android, and connects back to the same device.

What the app runs

The setup implementation restricts shell execution to these package-state operations:

  • pm disable-user --user 0 com.nothing.ntessentialspace
  • pm disable-user --user 0 com.nothing.ntessentialrecorder
  • The corresponding pm enable --user 0 commands during restoration.

The packages and their data are not deleted.

Android confirmations cannot be bypassed

You must reveal Developer Options, enable Wireless Debugging, open Android's pairing dialog, and supply the pairing code. Wireless Debugging can be disabled after setup.

If you uninstall without restoring Essential Space, the original Nothing handlers remain disabled. Restore first.
