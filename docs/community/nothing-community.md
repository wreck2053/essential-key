# Nothing Community post draft

## Title

Essential Key Remapper: single, double, and long-press actions without a PC

## Body

I built an open-source Android utility for supported Nothing and CMF phones that assigns separate actions to single, double, and long presses of the Essential Key.

It supports flashlight, camera, installed apps, URLs, media controls, Android navigation, screenshots, sound modes, and configurable HTTP requests for home automation.

Setup is completed on the phone using Android's local Wireless Debugging pairing flow—no desktop ADB or terminal is required. The app clearly documents that it disables the Essential Space and Essential Recorder handlers for the current user and provides a Restore Essential Space action that should be used before uninstalling.

Website and setup guide:
https://wreck2053.github.io/essential-key/
